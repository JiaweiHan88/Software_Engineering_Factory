# Tools — Scrum Master (Bob)

## Paperclip Skills

- `paperclip` — Heartbeat coordination, API calls, issue checkout, comments
- `para-memory-files` — PARA memory: facts, daily notes, knowledge graph, weekly synthesis

## BMAD Skills

- `bmad-sprint-planning` — Generate or update the sprint plan that sequences tasks for the full project
- `bmad-sprint-status` — Report current sprint progress and story statuses
- `bmad-create-story` — Prepare a story with all required context for developer implementation
- `bmad-correct-course` — Determine how to proceed when major changes are discovered mid-implementation
- `bmad-retrospective` — Review all work completed across an epic with lessons learned

## Copilot SDK Tools

- `sprint_status` — Read current sprint status, story lifecycle states
- `create_story` — Create a new story in the BMAD sprint system
