# Tools — Quick Flow Solo Dev (Barry)

## Paperclip Skills

- `paperclip` — Heartbeat coordination, API calls, issue checkout, comments
- `para-memory-files` — PARA memory: facts, daily notes, knowledge graph, weekly synthesis

## BMAD Skills

- `bmad-quick-flow-solo-dev` — Unified quick flow: clarify intent, plan, implement, review, present
- `bmad-dev-story` — Execute story implementation from spec file with TDD workflow
- `bmad-quick-dev` — Implement quick changes end-to-end (core of Quick Flow)
- `bmad-quick-spec` — Architect a quick but complete technical spec with implementation-ready stories
- `bmad-create-story` — Prepare a story with all required context for implementation
- `bmad-code-review` — Comprehensive adversarial code review with severity ratings

## Copilot SDK Tools

- `dev_story` — Execute story development workflow
- `create_story` — Create a new story in the BMAD sprint system
- `code_review` — Initiate code review workflow
- `sprint_status` — Read current sprint status, story lifecycle states
