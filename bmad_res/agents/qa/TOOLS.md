# Tools — QA Engineer (Quinn)

## Paperclip Skills

- `paperclip` — Heartbeat coordination, API calls, issue checkout, comments
- `para-memory-files` — PARA memory: facts, daily notes, knowledge graph, weekly synthesis

## BMAD Skills

- `bmad-code-review` — Comprehensive adversarial code review with severity ratings (LOW/MED/HIGH/CRITICAL)
- `bmad-review-adversarial-general` — General adversarial review across multiple quality facets
- `bmad-review-edge-case-hunter` — Find edge cases, boundary conditions, and overlooked scenarios
- `bmad-qa-generate-e2e-tests` — Generate end-to-end tests for existing features
- `bmad-testarch-atdd` — Acceptance Test-Driven Development methodology
- `bmad-testarch-automate` — Test automation framework design and implementation
- `bmad-testarch-ci` — CI/CD test integration pipeline design
- `bmad-testarch-framework` — Test framework architecture and setup
- `bmad-testarch-nfr` — Non-functional requirements testing (performance, security, reliability)
- `bmad-testarch-test-design` — Test case design patterns and strategies
- `bmad-testarch-test-review` — Review and validate existing test suites
- `bmad-testarch-trace` — Requirements traceability matrix and coverage mapping

## Copilot SDK Tools

- `code_review` — Initiate code review workflow
- `code_review_result` — Report code review findings
- `quality_gate_evaluate` — Evaluate quality gate pass/fail
- `sprint_status` — Read current sprint status, story lifecycle states
